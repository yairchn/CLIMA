SurfaceFluxes

SurfaceFluxes

Surface flux functions, e.g. for buoyancy flux, friction velocity, and exchange coefficients.

Byun1990

Compute surface fluxes using the approach in Byun (1990).

Plots

scales = [50,200,600,1000,10_000]

z_0 = 1.0
γ_m, γ_h = 15.0, 9.0
β_m, β_h = 4.8, 7.8
Pr_0 = 0.74

plot(Ri_range,
    [Byun1990.compute_exchange_coefficients(Ri,scale*z_0,z_0,γ_m,γ_h,β_m,β_h,Pr_0)[1]

source
CLIMA.SurfaceFluxes.Nishizawa2018.compute_friction_velocity — Function.
compute_friction_velocity(u_ave, θ, flux, Δz, z_0, a, Ψ_m_tol, tol_abs, iter_max)

Computes friction velocity, in Eq. 12 in Ref. Nishizawa2018, by solving the non-linear equation:

u_ave = ustar/κ * ( ln(Δz/z_0) - Ψ_m(Δz/L) + z_0/Δz * Ψ_m(z_0/L) + R_z0 [ψ_m(z_0/L) - 1] )

where L is a non-linear function of ustar (see compute_MO_len).

source
CLIMA.SurfaceFluxes.Nishizawa2018.compute_exchange_coefficients — Function.
compute_exchange_coefficients(z, F_m, F_h, a, u_star, θ, flux, Pr)

Computes exchange transfer coefficients:

  • K_D momentum exchange coefficient
  • K_H thermodynamic exchange coefficient
  • L_mo Monin-Obukhov length
